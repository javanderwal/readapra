# Statistical Publications Produced by APRA

This dataset contains a record of all statistical publications produced
by APRA that are available in `readapra`.

## Usage

``` r
apra_stat_pubs
```

## Format

A data frame with 7 rows and 3 variables:

- stat_pub_name:

  the full name of the statistical publication.

- stat_pub_acronym:

  the acronym of the statistical publication.

- cur_hist:

  available current and historical versions of the statistical
  publication.
